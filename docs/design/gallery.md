## 1. 场景总览

一座「回字形动线」的现代美术馆：观众从南侧入口进入**序厅**，左手西翼是**绘画长廊**，右手东翼是**雕塑与器物厅**，正前方深处是**影像与文献厅**。空间全部为室内白盒子，顶部有长条形天窗漫射自然光，每件展品头顶有一盏黑色轨道射灯，暖光洗亮展品、在墙面/地面落下柔和阴影。

- 渲染器：`antialias: true`，`outputColorSpace: SRGB`，`toneMapping: ACESFilmic`，`exposure 1.05`，`shadowMap: PCFSoft`
- 环境：`scene.background = #F4F1EA`；`FogExp2 #F4F1EA, density 0.008`（移动端 0.012）——远处墙面融入暖白，营造"无尽白"纵深
- 环境反射：drei `<Environment resolution={64}>` + 手写 `Lightformer`（顶部两条长形暖白 + 正面弱补光），**不依赖外部 HDR 文件**，离线可构建
- CSS 暗角：Canvas 容器叠 `radial-gradient(120% 90% at 50% 45%, transparent 62%, rgba(34,31,26,.14) 100%)`，`pointer-events: none`

**Animation**：场景挂载时雾密度从 0.02 → 0.008 缓动 2s（配合入场运镜，像眼睛适应展厅）；射灯在入场后按距入口由近及远每 80ms 一盏"点亮"（强度 0→目标 0.4s），呈"开馆"仪式感。

## 4. 灯光设计

三层照明，营造"均匀环境光 + 重点洗墙"的真实美术馆光环境：

| 层 | 光源 | 参数 |
|---|---|---|
| 基础 | HemisphereLight | 天 #FFF9EF / 地 #CFC4B0，intensity 0.55 |
| 天窗 | DirectionalLight（南向斜射，position [6,14,8]） | 色温 4000K `#FFF2E0`，intensity 1.1；**唯一全场投影光**：shadow-map 2048、相机包围盒覆盖 60×30m、bias -0.0004、radius 6（PCFSoft 软边） |
| 重点 | 每展品一盏 SpotLight（轨道位置 → 展品） | 3200K `#FFE3C2`；intensity 2.2（聚焦时 3.0）；angle 0.42rad（24°）；penumbra 0.5；distance 9；decay 1.6；**不投影**（中央维纳斯 + 序厅 2 巨幅 共 3 盏例外，map 512） |
| 补光 | 序厅中央 PointLight ×1（y 4.5） | #FFEEDD，intensity 0.35，distance 18，decay 2 —— 让角色在厅中央有立体感 |

**灯光预算与调度**：射灯组件常驻但 `visible` 按需 —— 常亮：中央维纳斯、标题墙上方洗墙灯；其余按"距玩家最近 6 盏"激活（每 0.3s 重算一次，切换时 0.4s 淡入淡出避免突变）。移动端上限 5 盏。
**假投影**：角色、展台、展柜、长凳下方各一张 `shadow-blob.png`（径向渐变），补偿不投影射灯的场景。
**聚焦光效**（展品被聚焦）：该展品射灯 intensity 2.2→3.0、angle 收窄 0.42→0.36rad，300ms 线性；画框上沿出现一道暖光洗墙渐变（自发光平面，opacity 0→.28）。

**Animation**：入场时射灯分组点亮（见 §1）；聚焦光效 300ms 线性插值；视频屏幕自发光随画面内容呼吸（强度 = 帧亮度均值 ×0.9，节流至 10Hz 更新）。

## 6. 展陈系统（五种 mount 构件规格）

所有构件由 `ExhibitRoot` 按配置驱动实例化；同几何共享 mesh，画框/展台/灯壳 InstancedMesh。

### 6.1 `wall-frame` 挂画（image）
- 画芯：`size.w × size.h` 平面，画芯中心高 1.55m（配置 position.y 控制）；微凸墙面 4cm
- 画框：`frame` 三色之一；黑/橡木 4cm 宽边深 5cm，鎏金 8cm 宽边深 7cm；内衬 2cm 米色卡纸边（#EFE9DC）
- **博物馆标签**：画框右下侧墙面，14×9cm 白色亚克力牌（y 1.35m），Canvas 纹理绘制：mono 编号（9px 等效）、作品名（衬线 600）、作者/年份（12.5px、灰）——走近 1.2m 内可清晰阅读
- 挂画倾角：上沿离墙 2°（真实挂画俯仰）

### 6.2 `pedestal` 展台（model）
- 白立方台座 0.5×0.5×1.05m（中央主展台 0.8×0.8×1.1m）
- 顶部展品 glb：加载中显示半透明占位胶囊；**无 src 时程序化回退雕塑**（球+环+多面体组合的白色"抽象雕塑"，与模板气质一致，保证 0 素材也能跑）
- `spin: true` 时绕 Y 自转 0.15 rad/s；标签牌立于台前地面 15° 斜面小座上

### 6.3 `vitrine` 展柜（image/model 小型）
- 白基座 0.6×0.6×0.8m + 玻璃罩 0.55×0.55×0.9m（顶部内嵌微型射灯，emissive 点）
- 内部：image 类型为 0.4m 高立式相片牌（15° 后仰）；model 类型为小型 glb

### 6.4 `screen` 壁挂屏（video）
- 16:9 屏幕 `size`（演示 2.4×1.35m），黑边框 6cm、深 8cm 贴墙，中心高 1.7m
- VideoTexture 静音循环播放；3m 外看不到内容时用 `poster` 静态图（性能）
- 屏下墙面对齐一条 1.2m 黄铜细线 + mono 编号

### 6.5 `panel` 信息面板（text/link）
- 1.2×0.9m 亚克力面板，四角黄铜铆钉，凸墙 2.5cm，中心高 1.5m
- text：白底，衬线标题 + 无衬线正文（Canvas 排版，自动换行，≤260 字）
- link：同款面板 + 黄铜描边 + 右上外链图标 + 底部「扫码/点击访问」提示行

**Animation（聚焦态，全部 300ms `easeOut`）**：画框描边自发光 0→0.15（#C89B5A）；展台/展柜基座前地面浮现金色地环（Ø1.1m 细环，opacity 0→.6 后 1.6s 呼吸脉冲 scale 1↔1.045）；屏幕亮度 +15%；面板铆钉微亮。标签牌在聚焦时自发光 0→.08 便于阅读。

## 8. 角色 Avatar

**造型**：美术馆人台式低多边形角色 —— 无面、无四肢细节的优雅剪影：胶囊躯干（肩宽 0.46，收至腰 0.36）+ 球体头部（Ø0.24，微后仰 3°）+ 一圈**黄铜颈环**（Ø0.13，模板唯一"角色个性"细节）+ 底部裙摆式圆台（Ø0.5，离地 0.05，暗示底座）。总高 1.75m。材质：哑光暖灰陶瓷 #B9B2A4 + 颈环黄铜。
**为什么不做骨骼动画**：模板要求"可复用、资源可替换"——程序化人台零外部依赖、永不穿模、步态用代码表达；`Avatar.tsx` 预留 `src` 字段可换任意 glb 角色。

**程序化动画**（全部代码驱动）：
| 状态 | 参数 |
|---|---|
| 待机 | 呼吸 scale y 1±0.008（2.4s 正弦）；头微侧 ±2°（5s 随机游走） |
| 行走 | 垂直起伏 y ±0.035m（步频 2.2Hz × 速度比）；前倾 4°；颈环随起伏反相延迟 40ms（惯性错觉） |
| 疾跑 | 起伏 ±0.05m / 3.4Hz；前倾 7° |
| 转向 | 身体朝向 lerp 角速度 10 rad/s，头部先行 15%（视线引导） |

碰撞胶囊：r 0.32，h 1.75；底部贴 `shadow-blob`（Ø0.9m）。第一人称下整身隐藏（含 blob）。

**Animation**：出生瞬间角色从地面"升起"就位（y -0.1→0 + opacity 0→1，0.5s `easeOutCubic`，入场运镜结束时触发）。

## 9. 相机系统

| 参数 | 第三人称（默认） | 第一人称 |
|---|---|---|
| 位置 | 角色头顶 1.55m 处为锚点，后撤 | 角色头内，眼高 1.62m |
| 距离 | 默认 3.4m，滚轮 1.6–6m（指数平滑） | — |

| 俯仰限制 | -60° ~ +75° | -80° ~ +80° |
| 阻尼 | 位置 lerp .12 / 帧；朝向由输入直接驱动 | 无位置阻尼（头部直驱） |
| 相机避障 | 锚点→相机方向 spherecast（r 0.25），命中则收至命中距 ×0.85，下限 0.6m；恢复时 lerp .15 防弹跳 | 无需 |

**模式切换**（`V` 键 / HUD 按钮）：0.6s `easeInOutCubic` 插值位置与 FOV；切换到第一人称时角色 0.2s 淡出（第三→第一）/ 淡入。切换瞬间保持 yaw/pitch 连续，不晕眩。
**聚焦运镜**（按 E 打开展品时）：相机 0.8s `easeInOutCubic` 滑至展品正前方 2.2m、中心高 1.55m 的"观赏位"（若已在 2.5m 内则仅转向对正），随后弹窗出现；关闭弹窗不强制归位（停在观赏位继续探索）。
**入场运镜 IntroDolly**：见 §12。

**Animation**：所有相机运动禁止线性——跟随用阻尼 lerp，叙事用 cubic 缓动；疾跑时 FOV +3（0.3s），营造速度感。

## 11. 碰撞系统

- 角色 = 胶囊（r 0.32）；障碍 = §2.1 墙体 AABB 列表 + 展台/展柜/长凳的圆柱或 AABB（由建筑/展陈组件注册到 `collision.ts`）
- 每帧：期望位移 → 逐障碍检测重叠 → 沿最小穿透轴推出（迭代 2 次）→ 得到滑墙效果；无重力、无跳跃（平整地面，y 恒 0）
- 性能：障碍总数 <40，逐一 AABB/圆检测开销可忽略；无需空间索引
- 门洞通行：门洞净宽 6m » 胶囊直径 0.64m，无卡门风险
- **Animation**：撞墙无镜头震动（美术馆气质 = 安静）；仅疾跑撞墙时速度立即归零并回弹 0.05m

## 12. 区域系统（zones.ts）

- 区域矩形同 §2 表；判定顺序 `[media, painting, sculpture, hall]`（翼厅优先，门洞重叠区归翼厅外即 hall）
- 玩家进入新区域：写入 store `zone` → HUD 展区名切换 + 小地图高亮（动效见 overlay.md §8）
- 首次进入某区域：触发一次"区域揭示"——该区域射灯组 0.4s 内依次点亮（开馆仪式的局部复用）
- **Animation**：区域名旧值 y-8 淡出 .25s → 新值 y+8 淡入 .4s（`easeOutCubic`）；小地图区域填充黄铜 12% 淡入 .3s

## 13. 交互系统（interaction.ts）

**聚焦判定（每 0.1s）**：候选 = 距离 < `focusRadius`（默认 2.6m）的展品 → 取「玩家朝向与展品方向夹角 < 55°」中**距离最近**者为聚焦对象（第一人称改用相机视线射线，命中即聚焦，更直觉）。
**聚焦表现**：§6 聚焦态光效 + 地环 + HUD 提示 chip（overlay.md §7）。
**打开详情**：`E` / 点击 / 点按 / 动作键 → 聚焦运镜（§9）→ store 进入 `modal` 态（3D 降帧 30fps、输入挂起）→ UI 层弹出展品弹窗（overlay.md §9）。
**视频展品特例**：弹窗打开时场景内屏幕暂停，弹窗内播放器带声音续播；关闭后场景屏幕从头循环（避免音画双源）。
**model 展品**：弹窗内不内嵌 3D 查看器（首版范围），正文提示"可环绕展台观看"。
**Animation**：聚焦 chip 出现 spring（stiffness 320, damping 26, y 12→0）；打开详情全链路时序：运镜 0.8s 与 chip 收起 0.2s 并行 → 0.5s 时抽屉开始滑入（重叠 0.3s，行云流水）。

## 14. 加载与入场衔接

1. `LoadingOverlay` 完成 → 点击「进入展厅」→ 覆盖层淡出 0.6s
2. **IntroDolly**：相机从入口上方 (0, 3.4, 9.5) 看向 (0,1.5,-2) → 2.8s `easeInOutCubic` 滑至第三人称默认位（角色身后 3.4m）；同帧角色就位动画（§8）、标题墙铜线生长（§3）
3. 运镜结束 → 输入解禁 → HUD 元件 stagger 淡入（间隔 0.08s，y 10→0，0.45s）
4. 移动端跳过运镜的"拖拽冲突"：运镜期间屏蔽输入即可

**Animation**：整条入场链 0.6 + 2.8 + 0.5 ≈ 4s；覆盖层淡出用 `easeInOutQuad`，运镜用 `easeInOutCubic`，HUD 用 spring。

## 15. 性能与降级实现要点

- 纹理：`useTexture` + `SRGBColorSpace`；展品图懒加载（Suspense 分组按翼厅），加载中画芯为米色卡纸占位
- glb：Draco 解码器本地 `/draco/`；`useGLTF.preload` 在 Loading 阶段并发拉取
- 视频：`VideoTexture`，`playsInline`，`muted loop`；离屏 >12m 时 `pause()`
- InstancedMesh：射灯外壳、画框（按框型 3 组）、展台、铆钉
- 每帧工作仅限：角色移动/碰撞、相机阻尼、聚焦判定（节流 0.1s）、灯光调度（0.3s）；**严禁**在 useFrame 中 setState（写 ref / store 瞬态字段）
- 降级（`navigator.hardwareConcurrency ≤4` 或移动端 UA）：pixelRatio 1.5、关射灯影、平行光影 1024、玻璃磨砂、雾 0.012、同屏激活射灯 5
- **Animation**：降级切换在加载期一次性判定，运行中不动态改（避免闪烁）；唯一例外——弹窗打开时 frameloop 降 30fps，关闭恢复 60（0.2s 内无感）
